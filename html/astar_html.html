A* Algorithm

UNIQUENESS:

Single source- Double Destinations

STEPS TO ATTAIN RESULT:

(i) Click the SOURCE button and select the cell you wish the source to be.

(ii) Then,click the DESTINATION1 button and choose one cell.

(iii) Then,click the DESTINATION2 button and choose one cell.

(iii) Click BLOCKS button to choose blocks in the grid.

(iv) By default Euclidean with weight one is chosen.

(v) By entering weight in the box you can change the weight and
can change the type by clicking manhattan.

(vi) Click START button. The resulting path will appear.
