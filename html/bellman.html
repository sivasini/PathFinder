Bellman Ford Algorithm

UNIQUENESS:

Multiple Source-Single Destination. The algorithm higlights all the shortest paths from all the selected source cells to the destination cell.

GUIDING TO RESULT:

(i) Click the SOURCE button and select a maximum
of 5 cells that you need to explore.

(ii)Click the DESTINATION button and select a
single cell that you want to reach.

(iii)Click BLOCKS button and select all the
cells that are to blocked and click START button

(iv)The result highlights paths from selected
source(s) to destination.
