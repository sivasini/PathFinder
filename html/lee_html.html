Lee's Algorithm

UNIQUENESS:

Single source-Multiple destination. The digits in the cells surrounding the choosen destination indicate the path count to reach that particular cell from source.

STEPS TO ATTAIN RESULT:

(i) Click the SOURCE button and select the cell you wish
the source to be.

(ii) Then,click the DESTINATION button and choose one cell
which will be destination.

(iii) Click BLOCKS button to choose blocks in the grid.

(iv) Click START button. The grid hightlight the path
between soruce and destination and prints the path count
fron source to the cells surrounding the destination.
